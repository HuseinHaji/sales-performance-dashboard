# Portfolio Copy: Sales Performance Dashboard

## Short Card Description

Built a synthetic sales performance analytics workflow that transforms transaction, customer, product, and target data into KPI summaries, margin analysis, and dashboard-ready outputs.

## Long Project Description

Sales Performance Dashboard is a portfolio implementation of a commercial analytics workflow for a simulated B2B sales organization. The project uses a synthetic sales dataset to model revenue, margin, products, customers, regions, sales channels, sales reps, and monthly targets.

The workflow generates raw data, validates quality checks, enriches transactions, calculates KPI tables, and exports a clean Power BI-ready dataset. It also includes PostgreSQL-style SQL analysis and a Streamlit executive dashboard prototype for management-style reporting.

This project is designed to demonstrate practical analyst work: turning raw transaction data into validated, business-readable outputs that support revenue monitoring, target tracking, product/customer analysis, and dashboard storytelling.

## Skills List

Python, pandas, numpy, SQL, KPI design, sales analytics, margin analysis, target tracking, Streamlit, Power BI-ready modeling, data validation, Docker, CI/CD, pytest.

## Business Value Bullets

- Tracks revenue, margin, growth, and target achievement.
- Identifies top-performing customers, products, regions, channels, and sales reps.
- Produces clean BI-ready datasets for dashboarding.
- Demonstrates commercial analytics from raw transactions to management reporting.
- Adds validation checks to improve reliability of reporting outputs.

## Recruiter-Friendly Impact Wording

Designed a reproducible sales analytics workflow that converts raw synthetic transaction data into validated KPI tables, performance summaries, and dashboard-ready exports.

## Disclaimer

This project uses a synthetic sales dataset for a simulated business case. It does not contain real company data and does not claim real business impact.
