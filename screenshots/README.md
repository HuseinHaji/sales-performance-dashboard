# Dashboard Screenshot Guide

Add these screenshots after running the Streamlit demo or building a Power BI report from `output/powerbi/powerbi_sales_dashboard.csv`:

- `executive_overview.png`: KPI cards, revenue trend, margin trend, month-over-month growth.
- `revenue_margin_analysis.png`: revenue by month, margin by month, channel revenue, discount impact.
- `product_performance.png`: product category revenue, top products by revenue and margin, product performance bands.
- `customer_performance.png`: top customers, customer value bands, revenue concentration, average order value.
- `target_achievement.png`: revenue vs target, margin vs target, above-target and below-target regions.

Use captions that clearly state this is a synthetic sales dataset and a portfolio implementation.
